# Cocktail Categorization Guide

This guide explains how to categorize cocktails in this collection.

## Category System Overview

Each cocktail should be tagged with **3-5 categories** across different dimensions:

### Required Categories (Every cocktail needs these)
1. **Base Spirit** - The primary alcohol
2. **Cocktail Family** - The structural template
3. **Flavor Profile** - The dominant taste

### Optional Categories (Add as relevant)
4. **Occasion** - When to drink it
5. **Technique** - How it's made
6. **Strength** - Alcohol content
7. **Region** - Cultural origin

## How to Tag a Cocktail

Add categories to the `# Kategorien` section using wiki-links:

```markdown
# Kategorien
- Base: [[Rum-Based]]
- Family: [[Sour]]
- Flavor: [[Citrus-Forward]]
- Occasion: [[Summer]]
- Technique: [[Shaken]]
- Strength: [[Standard ABV]]
- Region: [[Caribbean-Latin]]
```
